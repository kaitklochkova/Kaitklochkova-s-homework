{/bf Строки}
Типы читаются справа налево: char *str - имя переменной, * указатель, тип переменной. (= char str[])
Конец строки кодируется нулевым символом, нулевой символ не может встречаться в середине строки.
Читать строки можно: scanf("%s", str). Нет &!

{/bf Указатели.}
Тип данных.
Указатель -- число, адрес ячейки памяти (такого же размера, как и int).
Могут указывать на ячейки с другими указателями (списки, динам...)
*&a = a -- разыменование.
& -- Взятие адреса ячейки.
Есть стек вызовов, есть ОЗУ -- "куча".
Строка сама по себе указатель.
Если написать scanf("%s", &str), то значение строки запишется в её адрес.

В программе -- указатели на значения в "куче".
